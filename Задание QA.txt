Тестирование API 


Цель задания: Оценить навыки тестирования API на примере SWAPI


Задачи:


1. Протестируйте получение информации о персонажах "Luke Skywalker" и "Darth Vader" через SWAPI.

   * Проверьте, что запросы возвращают корректные данные о персонажах.
   * Убедитесь, что полученные данные содержат ожидаемую информацию (имя, рост, вес и т.д.).
   * Проверьте обработку ошибок при запросе несуществующего персонажа.

      2. Протестируйте получение информации о планетах, на которых побывали персонажи "Luke Skywalker" и "Darth Vader".


      * Убедитесь, что запросы возвращают корректные данные о планетах.
      * Проверьте соответствие полученных данных о планетах ожидаемой информации (название, климат, территория и т.д.).

         3. Напишите тест-кейсы для проверки функционала SWAPI.

            * Разработайте не менее 5 тест-кейсов для различных эндпоинтов SWAPI.
            * Укажите ожидаемые результаты, шаги выполнения и входные данные для каждого тест-кейса.


Инструкции:


Используйте любой удобный вам инструмент для тестирования API (например, Postman).

Предоставьте отчет о проведенном тестировании, включая найденные ошибки, результаты выполнения тест-кейсов и общую оценку качества API.



Ссылка на API https://swapi.dev/  (или https://www.swapi.tech/ )
